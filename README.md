# TASK 1 - Data Cleaning & Preprocessing

## Objective
Learn how to clean and preprocess raw data for Machine Learning using the Titanic Dataset.

## Operations Performed
- Imported dataset
- Checked null values and data types
- Handled missing values
- Encoded categorical variables
- Standardized numerical data
- Visualized outliers using boxplots
- Removed outliers using IQR method

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset Used
Titanic Dataset

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python preprocessing.py

## Output
Cleaned and preprocessed Titanic dataset ready for Machine Learning.
